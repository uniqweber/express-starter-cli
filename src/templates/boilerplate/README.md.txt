# {{projectName}}

This is a Domain-Driven Design (DDD) based Express Application starter, generated using `generate-expressts-starter`.

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Setup environment variables:
   Rename `.env.example` to `.env` and fill in the values.

3. Run the development server:
   ```bash
   pnpm dev
   ```

## Folder Structure

- `src/domains`: Feature-specific modules.
- `src/config`: Environment and service configuration.
- `src/libs`: Third-party library wrappers.
- `src/middlewares`: Express middlewares.
- `src/utils`: Helper functions.
- `src/shared`: Shared types and interfaces.
